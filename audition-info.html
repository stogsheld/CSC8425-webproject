Audition Information

A set of FAQs for auditioning can be found below.

Audition dates will be announced in early September - if you join our mailing list we'll give you more info closer to the time!

NUGSS are an inclusive society, and so we cast genderblind productions - so any character can be played by anyone! Therefore, you don't need to choose specific parts you'd like to play, but if you'd like to only be considered for certain roles, please tell us in the comments box of the audition form.

You should find out if you've landed a role within 5 working days of the final audition time.

Don't worry! Even if you're not cast, we'd love to have you in the chorus. There are often singing/speaking roles for chorus members too, so we'll prioritise people that auditioned for these parts.
